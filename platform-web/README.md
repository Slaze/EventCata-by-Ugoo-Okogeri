# EventCata Platform Web Scaffold

This directory is the first step of the refactor from the monolithic HTML app to a maintainable component-based web app.

## Goals
- Preserve route compatibility: `/`, `/detail`, `/create`
- Enable gradual migration of existing features from `index.original.html`
- Keep PWA/SEO metadata aligned with production shell

## Run

```bash
npm install
npm run dev
```
